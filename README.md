# Dr4g0n ROX

BookMyShow-style static movie ticket booking frontend project.

## Flow
index.html → home.html → movie.html → theatre.html → showtime.html → seats.html → snacks.html → payment.html → processing.html → ticket.html

## Features
- Professional responsive UI
- Movie listing, search and language filters
- Movie details page
- Theatre selection
- Showtime selection
- Interactive seat selection
- Snacks combo page
- Payment-style checkout UI
- QR-style ticket generation
- Booking history and profile pages

Open `index.html` in browser to start.
